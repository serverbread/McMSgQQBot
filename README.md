# McMSgQQBot
基于MiraiMC的用Skript编写的简单游戏内qq群消息互通插件

# 如何使用
编辑.sk文件，将on load事件中的{GROUP_ID}与{BOT_ID}设置为你qq机器人所要进行互通的群号以及你登录了miraimc的qq机器人的qq号
然后将其丢入/plugins/Skript/scripts

# 实现功能：
 - [x] QQ消息->游戏内
 - [X] 游戏内消息->QQ群
 - [X] 死亡消息转发
 - [X] 加入、退出消息转发
 - [ ] QQ群内命令执行
 - [ ] QQ群内查询服务器信息
 - [ ] QQ账号与游戏内账号绑定
